# Trainee Task  (FullStack)

## Цель

- Создать web-страницу согласно техническому заданию
- Процесс создания должен быть в GitHub (public repo)
- Web-страница должна был расположена в интернете

## Техническое задание

### Front-End

- Сверстать адаптивную страницу согласно [макету в Figma](https://www.figma.com/design/GRJbf19lk1BMSN8khHiUG8/%D0%A2%D0%B5%D1%81%D1%82%D0%BE%D0%B2%D0%BE%D0%B5-%D0%B7%D0%B0%D0%B4%D0%B0%D0%BD%D0%B8%D0%B5-%D0%B4%D0%BB%D1%8F-Fullstack-developers?node-id=0-1&p=f&t=Fje2h7xynOkgRORb-0)
    - Секция Hero будет содержать:
        - Видео на фоне должно запускаться автоматически при заходе на страницу в любом браузере.
        - Кнопка авторизации на сайте (только через Google oAuth2)
- Нельзя использовать фреймворки и библиотеки, строго JavaScript (или TypeScript), CSS
- Сборка проекта должна осуществляться с помощью [Vite](https://vite.dev/)
- Монеты криптовалют должны отображать реальную стоимость в режиме реального времени (Необходимо использовать публичный источник WebSockets)

### Back-End (На ваш выбор: Go, PHP, Rust, Python)

- Необходимо с помощью oAuth2 сделать Google авторизацию (работу с сессиями делать не нужно)
- На бекенде разрешено использовать необходимые вам библиотеки и фреймворки
- Endpoints на ваше усмотрение
- Базу данных использовать не нужно
